div and class

Div is used to point out a specific area in HTML, and class make divs recognizable.

Ex.

•HTML

<div class = “c1”>
This is class 1.
</div>

<div class = “c2”>
This is class 2.
</div>

•CSS
c1{
color : red;
}

c2{
color : orange;
}

Then, c1 and c2 will be red and orange, respectively. Note; there is a similar code; <div id = “something ”>. Id names cannot be used in HTML more than two times, whereas class names can be used as many times as you want.
